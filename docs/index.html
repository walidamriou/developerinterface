What is Developer Interface?

Developer Interface adds a task panel to the VS Code sidebar. It reads task definitions from .developerinterface/di-config-*.json, displays them as clickable items, and runs each command in the integrated terminal. You can create the config from a default template or generate it from a Makefile, and any save to the config is reloaded automatically in the view.

Quick Start

  1. Open your project in VS Code.
  2. Open Developer Interface view and create the template.
  3. Edit .developerinterface/di-config-project.json.
  4. then from the Developer Interface, run your tasks.

Minimal Config Example

{
  "version": "1.0.0",
  "title": "My Project",
  "tasks": [
    { "label": "Build", "command": "make build", "icon": "tools" },
    { "label": "Run", "command": "make run", "icon": "play" }
  ]
}
File name: .developerinterface/di-config-*.json
Useful icons: tools, play, testing, terminal
